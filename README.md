## Featured Projects

### Personal Second Brain / LLM Wiki
A local LLM-assisted knowledge system that keeps raw sources separate from compiled knowledge, using source summaries, project pages, indexing, provenance, and an append-only operations log so information compounds over time rather than being rediscovered in one-off chats.

### Microsoft 365 Service Desk Knowledge Workflow
A SharePoint-based service desk knowledge workflow designed around structured handoffs, Copilot-style grounding, review queues, and human approval. The aim is to turn repeated troubleshooting work into reusable, governed support knowledge.

### Meeting Notes And Content Ingest Automation
Repeatable workflows for turning transcripts, notes, and source material into structured summaries, actions, links, and reusable outputs.

### Automation And Dashboard Workflows
Workflow concepts using Power Automate, Power Query, Excel, Google Sheets, and structured review flows to reduce repeated manual admin work and improve operational visibility.
